# react-cli ![GitHub](https://img.shields.io/github/license/mashape/apistatus.svg)

参考vue-cli 的 init 功能，初始化 react 项目
### 用法
```bash
# 全局安装

$ npm i -g react-cli

# 初始化项目

$ react-cli init `project-name`

```

### 发布命令
```bash
$ npm publish 
# 发布代码

$ npm unpublish <package>@<version>
# 可以撤销发布自己发布过的某个版本代码。
```
